# PDF Compressor

Простая утилита для сжатия PDF-файлов на Python. Уменьшает размер документа за счёт пересжатия изображений и оптимизации структуры файла. Работает через графический интерфейс и командную строку.

## Возможности

- Графический интерфейс (Windows, macOS, Linux)
- Командная строка для автоматизации и пакетной обработки
- Три уровня сжатия: низкий, средний, высокий
- Отображение исходного и итогового размера файла

## Требования

- Python 3.10+
- [PyMuPDF](https://pypi.org/project/PyMuPDF/) (`pymupdf`)

## Установка

```bash
git clone https://github.com/<username>/pdf-compressor.git
cd pdf-compressor
python -m pip install -r requirements.txt
```

## Использование

### Графический интерфейс

```bash
python main.py
```

На Windows можно запустить двойным кликом по `compress.bat`.

1. Выберите исходный PDF-файл
2. Укажите путь для сохранения (необязательно)
3. Выберите уровень сжатия
4. Нажмите **Сжать PDF**

По умолчанию результат сохраняется как `имя_compressed.pdf` рядом с исходным файлом.

### Командная строка

```bash
# Сжатие со средним уровнем (по умолчанию)
python main.py document.pdf

# Высокое сжатие
python main.py document.pdf -l high

# Указать путь для сохранения
python main.py document.pdf -o compressed/document.pdf -l medium
```

#### Параметры

| Параметр | Описание |
|----------|----------|
| `input` | Путь к исходному PDF |
| `-o`, `--output` | Путь для сохранения результата |
| `-l`, `--level` | Уровень сжатия: `low`, `medium`, `high` |

## Уровни сжатия

| Уровень | Описание | Качество JPEG | Целевой DPI |
|---------|----------|---------------|-------------|
| `low` | Минимальная потеря качества | 85 | 150 |
| `medium` | Баланс размера и качества | 65 | 120 |
| `high` | Максимальное сжатие | 45 | 96 |

## Ограничения

- Наибольший эффект достигается на PDF с изображениями (сканы, фото, презентации)
- Текстовые PDF без картинок могут почти не уменьшиться — они уже компактны
- Исходный файл не изменяется; создаётся новая копия
